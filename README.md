# URL Shortener
A production-ready URL shortener with analytics, QR codes, and user authentication. Built to demonstrate full-stack development skills with modern web technologies.


## Key Features

URL Shortening with custom alias support
QR Code Generation for each shortened link
Real-time Analytics - track clicks, devices, and geographic data
User Authentication with protected routes
Responsive UI built with Shadcn UI components



## Tech Stack
Frontend: React 18, React Router v6, Tailwind CSS, Shadcn UI
Backend: Supabase (Auth, Database, Storage)
Database: PostgreSQL with Row-Level Security


## Quick Start
Clone and install
git clone <repo-url>
cd url-shortener
npm install

## Environment setup
cp .env.example .env
### Add your Supabase credentials

## Run development server
npm run dev
